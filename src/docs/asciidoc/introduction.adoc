해당 문서는 Race-On API에 대한 문서입니다.

API 사용법 관련 문의는 Discord DM을 통해 주시면 최대한 빨리 답변드리겠습니다.

=== Base Format
[source,json]
----
{
  "code": 200,
  "message": "응답 메시지",
  "data": {
    "key1":"value1",
    "key2":"value2"
  },
  "success": true
}
----
|===
|Name |Description | optional

|success | 성공여부 | false
|code | 응답 코드(http status code) | false
|data | 응답 데이터(없을 경우 생략 가능) | true
|message | 응답 메시지(성공/실패 원인을 간략히 설명) | false
|===

*code, message*

code는 일반적으로 http status code를 따르고, message는 실패 원인을 간략히 설명합니다.

(성공시 "요청에 성공했습니다.","생성에 성공했습니다."를 반환합니다.)

code와 message를 통해 API문서에서 실패 원인을 확인하고 개발하시면 됩니다.


*data* :

key, value 형식으로 data를 반환하며 반환할 데이터가 없을 경우 아래와 같이 생략될 수 있습니다.

실패의 경우 대부분 data가 생략됩니다.

[source,json]
----
{
  "code": 200,
  "success": true
}
----
