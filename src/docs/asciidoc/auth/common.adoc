=== Social Login 구현(SDK 사용)

1. SDK를 통해 로그인을 진행하고 받은 응답에서 id_token을 추출합니다.

2. id_token과 SoicalProvider를 Body로 넣어 로그인을 시도합니다.

3. 회원가입된 유저인 경우 JWT가 반환됩니다.

4. 회원가입되지 않은 유저인 경우 에러 응답을 반환합니다. 이 경우 회원가입 API 로 이어나가면 됩니다.

image::./images/oauth_process.png[]

여기서 OAuth Server란 Kakao, Google, Apple과 같은
인증 서비스 제공자의 서버를 의미한다.
