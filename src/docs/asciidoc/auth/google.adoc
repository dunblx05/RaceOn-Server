===== RequestURI
```
'https://accounts.google.com/o/oauth2/v2/auth?client_id=${CLIENT_ID}&redirect_uri=${REDIRECT_URI}&response_type=${RESPONSE_TYPE}&scope=${SCOPE}'
```

===== Query String
[%header,cols="2,4"]
|===
|Name         | Value

|RESPONSE_TYPE| code
|SCOPE| profile
|===
